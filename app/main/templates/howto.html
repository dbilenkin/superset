How to Play

How to make a set

Cards have different attributes like

shape, number, pattern, color and background

To make a set each of these attributes have to
be all the SAME or all DIFFERENT

This is a set because it has:

All the SAME shape
All the SAME pattern
All the SAME color
All DIFFERENT numbers

How NOT to make a set

If the attributes are not all the SAME or all DIFFERENT then it's not a set.

This is NOT a set even though it has:

All DIFFERENT shapes
All the SAME pattern
All the SAME number

Because two cards are red and one is green.

4 and 5 card sets

Sets are not limited to 3 cards.
The same rule of all the SAME or all DIFFERENT
apply to 4 or 5 cards as well

This is a set because it has:

All DIFFERENT shapes
All DIFFERENT patterns
All DIFFERENT colors
All the SAME number
